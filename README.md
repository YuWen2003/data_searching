# data_searching
Please design a program so that users can read a file "Input.txt". There are 50 data in the file, each data includes name, ID, math score, English score, and meet the following requirements.
(1) This program must contain a structure variable, which includes name, ID, math score, and English score.
(2) After inputting the ID, the information of the ID can be printed out and the average score can be calculated.
(3) If the ID is entered incorrectly, an error message should be printed out.
(4) The program should be able to be executed repeatedly.
